About Course

Explore the Core Clusters of Algorithmic Learning

Foundations

This cluster focuses on establishing a strong foundation in algorithm design and analysis. It covers the fundamental principles of algorithmic thinking, including understanding design philosophies and developing key intuitions for solving computational problems. Students will learn about space and time complexities, which are critical for evaluating an algorithm's efficiency, and the concept of order of an algorithm to determine its scalability. The cluster also introduces common problem-solving patterns such as recursion, iteration, and backtracking, providing the tools to approach a wide range of algorithmic challenges systematically.

Structured Data

This cluster explores advanced data structures for efficient data organization and retrieval, including graphs, trees, Tries, AVL Trees, 2-3 Trees, and Red-Black Trees, along with traversal techniques like DFS and BFS. It covers Heaps for priority queues, Array Queries, and advanced tools like Sparse Tables, Segment Trees, Fenwick Trees, and Skip Lists for range queries and updates. These structures and techniques enable efficient handling of complex data operations and problem-solving.

Sorting

This cluster focuses on the fundamental algorithms and techniques for sorting and searching, which are critical to efficient data processing and retrieval. Students will explore various sorting methods, such as comparison-based algorithms and specialized techniques, along with optimized searching strategies. These devices are essential for organizing data and enabling quick access, forming the basis for many advanced computational applications.

Graph Algorithms

This cluster focuses on essential graph algorithms, including techniques for finding the shortest path and constructing spanning trees. Students will learn algorithms like Dijkstra's, Bellman-Ford, and Floyd-Warshall for shortest paths, as well as Kruskal's and Prim's algorithms for spanning trees. These concepts are fundamental for solving real-world problems in networks, optimization, and connectivity.

// Function to perform Dijkstra's algorithm
void dijkstra(int source, int destination, vector> &graph, const vector &stops) {
    int n = graph.size(); // Number of stops
    vector distance(n, INT_MAX); // Distance from source to each stop
    vector parent(n, -1);        // To store the path
    priority_queue, vector>, greater>> pq;

    // Initialize source distance and push it into the priority queue
    distance[source] = 0;
    pq.push({0, source});

    while (!pq.empty()) {
        int currentDistance = pq.top().first;
        int currentStop = pq.top().second;
        pq.pop();

        // If the destination is reached, stop the search
        if (currentStop == destination) break;

        // Process neighbors
        for (const Edge &edge : graph[currentStop]) {
            int nextStop = edge.destination;
            int weight = edge.weight;

            if (distance[currentStop] + weight < distance[nextStop]) {
                distance[nextStop] = distance[currentStop] + weight;
                parent[nextStop] = currentStop;
                pq.push({distance[nextStop], nextStop});
            }
        }
    }

    // Output the shortest distance
    cout << "Shortest Distance from " << stops[source] << " to " << stops[destination] << ": " << distance[destination] << " kms.\n";

    // Calculate and display the estimated time in minutes
    if (distance[destination] != INT_MAX) {
        double time = static_cast(distance[destination]) / 30.0 * 60.0; // Time in minutes (converted from hours)
        cout << "Estimated Time to Reach: " << fixed << setprecision(2) << time << " minutes.\n";

        // Display the path
        cout << "Path: ";
        int current = destination;
        vector path;
        while (current != -1) {
            path.push_back(current);
            current = parent[current];
        }
        for (auto it = path.rbegin(); it != path.rend(); ++it) {
            cout << stops[*it] << (it + 1 != path.rend() ? " -> " : "\n");
        }
    } else {
        cout << "No path exists between the selected stops.\n";
    }
}

// Function to compute the LPS (Longest Prefix Suffix) array for KMP algorithm
vector computeLPSArray(const string &pattern) {
    int m = pattern.length();
    vector lps(m, 0);
    int length = 0; // Length of the previous longest prefix suffix
    int i = 1;

    while (i < m) {
        if (pattern[i] == pattern[length]) {
            length++;
            lps[i] = length;
            i++;
        } else {
            if (length != 0) {
                length = lps[length - 1];
            } else {
                lps[i] = 0;
                i++;
            }
        }
    }
    return lps;
}

// KMP String Matching Algorithm
bool KMPSearch(const string &text, const string &pattern) {
    int n = text.length();
    int m = pattern.length();
    vector lps = computeLPSArray(pattern);

    int i = 0; // Index for text
    int j = 0; // Index for pattern
    while (i < n) {
        if (pattern[j] == text[i]) {
            i++;
            j++;
        }
        if (j == m) {
            return true; // Pattern found in text
        } else if (i < n && pattern[j] != text[i]) {
            if (j != 0) {
                j = lps[j - 1];
            } else {
                i++;
            }
        }
    }
    return false;
}

void searchTickets(const vector &tickets, const string &query) {
    cout << "Search Results for \"" << query << "\":\n";
    bool found = false;

    for (const auto &ticket : tickets) {
        if (KMPSearch(ticket.ticketID, query) || KMPSearch(ticket.userName, query) || KMPSearch(ticket.travelDate, query)) {
            found = true;
            cout << "Ticket ID: " << ticket.ticketID << "\n"
                 << "User Name: " << ticket.userName << "\n"
                 << "Travel Date: " << ticket.travelDate << "\n"
                 << "Status: " << ticket.status << "\n\n";
        }
    }

    if (!found) {
        cout << "No tickets match the search query.\n";
    }
}
